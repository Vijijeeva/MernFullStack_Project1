This is a MERN stack project, that takes excel files as input and gives output, various visualization techniques that includes line chart, bar graph, box plot etc... 
The project offers a convenient way to analyze and present data visually, making it easier for users to interpret and understand the information.
The project aims to provide users with a user-friendly approach to analyze and present data visually, facilitating easier interpretation and understanding of the information.

# Overview

The Excel Analytics Platform is a data processing and visualization application designed to upload Excel files and convert raw data into meaningful insights.
The goal of this project is to help users analyze data without needing Excel formulas or advanced coding knowledge.
Users can:
* Upload .xlsx / .csv files
* Clean and transform data
* Generate dynamic visual charts
* Export analytics reports
This project is ideal for business users, analysts, and students who need a fast data analysis tool.

# Core Objectives
1. Convert Excel spreadsheets into structured analytics dashboards
2. Reduce manual work—no complex Excel formulas or macros required
3. Provide a simple and user-friendly interface
4. Automate chart generation and reporting

# Tech Stack 
# Backend
1. Python (FastAPI / Flask)
2. Pandas — Data cleaning & processing
3. OpenPyXL — Excel reading/writing
4. NumPy — Numerical analysis
5. React.js
6. TailwindCSS / Bootstrap UI
7. Chart.js / Recharts for graphs

# Storage
1. MongoDB / Firebase / SQLite
2. Local storage for development

# Features
# 1. File Upload
1. Upload .xls, .xlsx, or .csv
2. Validates column formatting
3. Detects duplicates and missing values

# 2. Data Cleaning Tools
1. Remove duplicate rows
2. Auto-fill missing values
3. Convert column types

# 3. Data Visualization
1. Automatic chart generation based on column type
2. Line charts, bar graphs, pie charts, Whisker chart, Bubble chart, 3D surface chart
3. Custom filters (date, category, numeric range)

# 4. Export
1. Export cleaned data into new spreadsheet
2. Export charts into PNG/PDF
